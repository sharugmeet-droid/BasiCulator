BasiCulator is a **user-friendly Dart calculator** that performs basic arithmetic and comparison operations on two numbers.

---

## Features

### Arithmetic Operations
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`) with zero check
- Modulus (`%`)

### Comparison Operations
- Greater than (`>`)
- Less than (`<`)
- Equal to (`==`)

---

## How It Works

1. Prompts the user to **enter two numbers**.
2. Performs **arithmetic operations** and displays the results.
3. Performs **comparison operations** and displays boolean results.
4. Handles division by zero with an error message.

---
